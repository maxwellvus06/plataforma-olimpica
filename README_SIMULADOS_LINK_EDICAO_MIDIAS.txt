# Ajustes de Simulados — link público, edição e mídias

Alterações principais:

1. Link público visível
- Simulados marcados como públicos agora exibem uma caixa com o link.
- Botões: copiar link e abrir link público.
- O mesmo vale para simulados gerados pelo Banco de Questões.

2. Edição antes do início
- ADM, Staff, Monitor e Professor/Orientador podem editar o simulado enquanto ele ainda não chegou à data/hora de início.
- Depois do início, a edição fica bloqueada para proteger histórico, envios e ranking.
- A equipe pode pré-visualizar o simulado quantas vezes quiser, sem contar como tentativa.

3. Pré-visualização da equipe
- Botão “Pré-visualizar” para ADM, Staff, Monitor e Professor/Orientador.
- Abre o ambiente de exibição sem gerar envio nem tentativa.

4. Questões manuais com mídias
- Questões manuais no cadastro de simulado agora aceitam imagens, PDF e DOCX.
- As mídias aparecem junto da questão no ambiente de resolução/pré-visualização.

5. Observação
- As regras do Firestore/Storage deste pacote são mantidas da versão anterior.
- Caso o upload de mídia falhe, confirme as regras do Storage.
