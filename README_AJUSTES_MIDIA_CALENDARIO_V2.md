# Ajustes aplicados

- Mídias e arquivos passam a abrir em visualizador interno da plataforma.
- Aulas abrem em ambiente interno com comentários.
- Painéis grandes de cadastro/importação ficam retráteis.
- Calendário Oficial passa a usar Data início e Data final no cadastro manual e no modelo/importação Excel.
- Simulados recebem pré-visualização para equipe, edição antes do início e questões manuais com anexos.

Arquivos principais: `index.html` e `app.js`.
