NÍVEL VISUALIZADOR

Implementado novo nível de acesso: Visualizador.

Permissões:
- Dashboard Analítico: visualizar apenas dados do escopo concedido.
- Calendário Oficial: visualizar.
- Importar/Resultados: visualizar resultados do escopo concedido.
- Relatórios Comparativos: gerar relatórios apenas do escopo concedido.
- Alunos: visualizar alunos das escolas/cidades concedidas.
- Olimpíadas: visualizar.

Cadastro:
Na aba Gerenciar Usuários, o ADM pode criar usuário do nível Visualizador.
Ao escolher Visualizador, aparece o bloco "Escopo do Visualizador".
O ADM pode selecionar uma ou várias cidades e/ou uma ou várias escolas.

Regra de escopo:
- Cidade selecionada libera todas as escolas/resultados/alunos daquela cidade.
- Escola selecionada libera apenas aquela escola.
- É possível combinar cidades inteiras com escolas específicas.

Edição:
Se o usuário já for Visualizador, o modal de edição mostra seletores múltiplos para atualizar cidades e escolas permitidas.
