# Correção — botões de Exportar Excel

Esta versão recoloca os botões de **Exportar Excel** em:

- Resultados;
- Calendário Oficial;
- Olimpíadas Disponíveis.

A exportação respeita os filtros aplicados na tela em Resultados e Calendário.

Permissão de geração:

- ADM;
- Staff;
- Gestor;
- Escola.

Não houve alteração em Firestore Rules ou Storage Rules.
