# Ajustes do módulo Simulados

- Formato reorganizado:
  - Objetivo — anexar prova pronta
  - Dissertativo — anexar prova pronta
  - Misto — anexar prova pronta
  - Criar prova manualmente
- Modo manual agora usa questões digitadas na própria plataforma.
- Imagens podem ser coladas no enunciado com Ctrl+V; elas são enviadas para o Firebase Storage e aparecem abaixo da questão.
- Anexos/mídias das questões não abrem mais em outra aba; aparecem dentro do ambiente do simulado.
- Ambiente de prova refeito com área de questões maior e área de respostas maior.
- Questões dissertativas agora têm resposta e anexo por questão.
- ADM, Staff, Gestor, Professor/Orientador e Monitor mantêm acesso de gestão/preview.
