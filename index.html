Configuração Firebase necessária

Para o chat em tempo real funcionar, você precisa configurar um projeto Firebase gratuito. Siga os passos abaixo:

  1. Acesse console.firebase.google.com e crie um projeto gratuito
  2. Ative o Realtime Database em modo de teste
  3. Vá em Configurações do Projeto → Seus apps → Adicione um app Web
  4. Copie as credenciais e cole na função initFirebase() no app.js

Enquanto não configurado, as salas ficam visíveis mas o chat não conecta.
