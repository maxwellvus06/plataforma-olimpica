EDITOR DE LAYOUT PLUS

1. Substitua index.html, app.js e database.js no GitHub.
2. No Firebase Console, vá em Build > Storage > Rules.
3. Cole o conteúdo de storage.rules e clique em Publicar.
4. O erro storage/unauthorized no upload de banner/logo/fundo é regra de Storage não publicada ou Storage não ativado.

Novos recursos do Editor de Layout:
- Estilo visual: clássico, glass, minimalista, institucional, olímpico.
- Densidade: compacta, confortável, ampla.
- Arredondamento: reto, suave, médio, forte.
- Fonte: sistema, limpa, serifada, mono.
- Fundo: sólido, gradiente ou imagem.
- Imagem de fundo da plataforma.
- Banner com altura e posição configuráveis.
- Continua com layout por escopo: global, nível, cidade, escola e usuário.

Observação: esta versão mantém as regras abertas para teste. Em produção, feche as regras.
