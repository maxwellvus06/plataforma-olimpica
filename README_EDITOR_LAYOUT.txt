EDITOR DE LAYOUT - COMO USAR

1. Substitua index.html, app.js e database.js no GitHub.
2. Suba também firestore.rules e storage.rules no Firebase, se estiver usando regras de teste.
3. Entre como ADM.
4. No menu lateral, abra Editor de Layout.
5. Edite textos, cores, paleta, logo e banner.
6. Clique em Salvar layout.

ONDE FICA SALVO
- Textos e cores: Cloud Firestore > sistema_layout > config
- Logos e banners enviados: Firebase Storage > layout/

IMPORTANTE
- Este editor não altera módulos, banco de dados, permissões nem lógica do sistema.
- Ele edita apenas identidade visual, cores principais, logo e banner.
- Não usa localStorage nem sessionStorage.
